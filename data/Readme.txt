# Datos

Los datos utilizados en este análisis no se incluyen en el repositorio.

Para reproducir el análisis se requieren los siguientes archivos:

- Beneficio_Neto_Total.tif
- Tremarctos_ornatus.tif
- IHEH_2018.tif
- Complejos_de_Paramos_Escala100k.shp

Estos datos deben almacenarse en la carpeta `data/` antes de ejecutar el script.
